# Estad_Inferencial

Bienvenido/a al repositorio del curso Estadística Descriptiva para Economía. Aquí encontrarás materiales, guías y recursos que te ayudarán a comprender cómo organizar, resumir e interpretar datos económicos de manera rigurosa. La estadística descriptiva es el primer paso para convertir información en evidencia útil para el análisis económico, y este repositorio será tu apoyo durante el semestre.

En el siguiente link encontraras el contenido programatico para este semestre.

https://www.notion.so/Estad-stica-Inferencial-312f76670a18803f87dfda78e2767930?source=copy_link

En este semestre utilizaremos de manera abierta Inteligencia Artificial (IA), en este caso el curso cuenta con "Cuaderno de Notas" mediado por IA en Notebook LM. Los estudiantes del curso pueden interactuar con un chat alimentado por textos académicos utilizados en mis clases, se recomienda limitar las preguntas a temas relacionados con estadística descriptiva, debido a que se encuentra ajustado para responder lo que se encuentra dentro de los insumos proporcionados.
